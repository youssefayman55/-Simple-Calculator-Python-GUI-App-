# -Simple-Calculator-Python-GUI-App-
A modern desktop calculator built using Python and Tkinter. It provides a clean graphical interface for performing basic arithmetic operations.

🚀 Project Description

This project is a GUI-based calculator application that allows users to:

1- ➕ Addition

2- ➖ Subtraction

3- ✖️ Multiplication

4- ➗ Division

5- % Modulus

🧮 Real-time expression evaluation

It demonstrates how to build a functional desktop application using Python.

🛠️ Tech Stack:

1- Python 🐍

2- Tkinter (GUI framework) 🖥️

3- Built-in eval() function ⚙️

▶️ How to Run:

1- Save the file as calculator.py

2- Run it using: python calculator.py

🎯 Features: 

1- 🧮 Basic Arithmetic

2- Supports +, -, *, /

3- Percentage (%) operation

4- Decimal numbers support

🖥️ GUI Interface:

1- Modern dark-themed UI

2- Button-based input system

3- Large display screen for results

4- ⚡ Real-Time Calculation

5- Expression is built dynamically

6- Uses Python eval() for instant computation

7- 🧹 Clear Function

7- AC button resets the entire expression

🧠 Concepts Used: 

1- GUI development with Tkinter

2- Event-driven programming

3- String manipulation

4- Global variables

5- Expression evaluation using eval()

💡 Future Improvements:

1- 🔢 Scientific calculator (sin, cos, tan)

2- 🧾 Calculation history feature

3- 🎨 Modern UI redesign (custom buttons & animations)

4- 📱 Mobile version using Kivy

5- 🖥️ Convert to desktop app (.exe)

⚠️ Notes:

1- eval() is used for simplicity (not recommended for secure production apps)

2- Input is built using button clicks only

3- Works offline (no internet required)
